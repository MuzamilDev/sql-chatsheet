S Q L C O M M A N D S

Data Defination Language

  • CREATE TABLE t

    • This sql command create a new table

  • DROP TABLE t

    • This sql command Remove a Table

  • ALTER TABLE t ADD col-name value

    • This Sql command add new column in table

  • ALTER TABLE t MODIFY col-name value

    • This sql command modify existing column

  • TRUNCATE TABLE t

    • This sql command remove all data from rows

DATA MANUPLATION LANGUAGE

  • INSERT INTO t col-name VALUE

    • This sql command insert data in columns of table

  • UPDATE t SET col-name WHERE condition

    • This sql command update value in specific column with where condition (i,e usually it is primary key)

  • DELETE FROM t WHERE condition

    • This sql command delete column of table with condition (i,e usually it is primary key)

DATA ACCESS

  • SELECT * FROM t
  • SELECT a FROM t
  • SELECT a,b FROM t WHERE condition
  • SELECT DISTINCT a FROM t WHERE condition
  • SELECT a,b FROM t ORDER BY ASC/DESC
  • SELECT a,b FROM t LIMIT numbers
  • SELECT TOP number FROM a,b
